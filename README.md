# Warehouse Management (WMS) ל-ERPNext

אפליקציית WMS המתווספת מעל מודול ה-Stock של ERPNext.
מוסיפה שכבת מיקומים פיזיים (אזור ← מדף ← קוד מיקום), איחסון מנוהל,
ליקוט, וסריקת ברקוד.

## התקנה
```
bench get-app https://github.com/israrobots-oss/warehouse_mgmt.git
bench --site <site> install-app warehouse_mgmt
```
